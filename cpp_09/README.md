## 📘 解説

### 🔹 概要
各課題では **コンテナクラスを1つ以上使って** 問題を解きます。  
**一度使ったコンテナクラスは他の課題で使えません。**

使用可能な C++98 コンテナクラス一覧：

#### 配列系（順序あり）:
- `std::vector`
- `std::deque`
- `std::list`

#### 連想系（キー → 値）:
- `std::map`
- `std::multimap`
- `std::set`
- `std::multiset`

⚠️ コンパイルフラグは以下を使用してください：
```sh
-Wall -Wextra -Werror -std=c++98
```
